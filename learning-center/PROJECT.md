# Learning Center Project Charter

## Problem
Research context is getting lost across chats, sessions, and agents.

## Solution
Build a structured learning center that turns research into:
- persistent source-backed memory
- reusable curricula
- drills and evaluation rubrics
- onboarding kits for agents and people

## Scope (Phase 1)

- Shared data model (sources + notes)
- Track-based curriculum folders
- Context-bundle generator
- Operational scripts for adding and querying knowledge

## Scope (Phase 2)

- Search and ranking over notes
- Auto-ingest connectors (YouTube metadata, docs, repos)
- Agent prompt injection from generated context bundles
- Public-facing learning modules for external trainees

## Success metrics

- Context loss reduced (handoffs use context bundle)
- Time-to-ramp for new agent under 10 minutes
- At least 1 teachable module/week per active track
- Observable improvement in build quality and speed
