# Interview Voice System (Gauntlet Track)

Purpose: build a consistent, high-signal interview voice that demonstrates AI-first engineering competence under pressure.

## Components

- `voice-manifesto.md` — tone, principles, and communication rules
- `answer-library-v1.md` — reusable answer structures and model responses
- `mock-script-v1.md` — realistic mock prompts + scoring rubric

## Usage

1. Read manifesto before every mock.
2. Use answer-library structures (60s + 3m versions).
3. Run mock script and score each response.
4. Update answers with concrete project evidence.
